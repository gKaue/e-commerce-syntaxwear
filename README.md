 # SyntaxWear - Projeto de E-commerce Hipotético
    2 
    3 Bem-vindo ao repositório do projeto **SyntaxWear**!
    4 
    5 Este é um projeto de e-commerce hipotético desenvolvido com o propósito de **exercitar e aprimorar habilidades em desenvolvimento web front-end**.
    6 
    7 ## Sobre o Projeto
    8 
    9 O **SyntaxWear** simula uma loja online de tênis e moda urbana, com foco principal na criação de uma interface de usuário responsiva e visualmente atraente. Este projeto é uma oportunidade para explorar e 
      aplicar conceitos de HTML, CSS e JavaScript, bem como práticas de design responsivo para garantir uma excelente experiência em diferentes dispositivos.
   10 
   11 ## Funcionalidades (Atuais/Pretendidas)
   12 
   13 -   **Navegação intuitiva:** Menu de navegação claro e de fácil uso.
   14 -   **Seção Hero:** Destaque visual para produtos ou promoções.
   15 -   **Categorias de Produtos:** Exibição organizada de diferentes tipos de tênis.
   16 -   **Grade de Produtos:** Layout para apresentar produtos em destaque ou mais vendidos.
   17 -   **Responsividade:** Design adaptável a telas de desktop, tablets e dispositivos móveis. (Incluindo a rolagem horizontal para categorias em telas menores!)
   18 
   19 ## Tecnologias Utilizadas
   20
   21 -   **HTML5:** Estrutura semântica do conteúdo.
   22 -   **CSS3:** Estilização e responsividade (com Flexbox, Media Queries e outras técnicas).
