# SyntaxWear
  
Bem-vindo ao repositório do projeto **SyntaxWear**!
    
Este é um projeto de e-commerce hipotético desenvolvido com o propósito de **exercitar e aprimorar habilidades em desenvolvimento web front-end**.
    
## Sobre o Projeto

O **SyntaxWear** simula uma loja online de tênis e moda urbana, com foco principal na criação de uma interface de usuário responsiva e visualmente atraente. Este projeto é uma oportunidade para explorar e 
      aplicar conceitos de HTML, CSS e JavaScript, bem como práticas de design responsivo para garantir uma excelente experiência em diferentes dispositivos.

## Funcionalidades (Atuais/Pretendidas) 
-   **Navegação intuitiva:** Menu de navegação claro e de fácil uso.
-   **Seção Hero:** Destaque visual para produtos ou promoções.
-   **Categorias de Produtos:** Exibição organizada de diferentes tipos de tênis.
-   **Grade de Produtos:** Layout para apresentar produtos em destaque ou mais vendidos.
-   **Responsividade:** Design adaptável a telas de desktop, tablets e dispositivos móveis. (Incluindo a rolagem horizontal para categorias em telas menores!)

## Tecnologias Utilizadas

**HTML5:** Estrutura semântica do conteúdo.
**CSS3:** Estilização e responsividade (com Flexbox, Media Queries e outras técnicas).
